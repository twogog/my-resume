# Resume Landing Page

Статичный сайт-визитка под GitHub Pages на основе резюме.

## Что внутри

- `index.html` — основная страница
- `styles.css` — стили и адаптивная верстка

## Как открыть локально

Просто открой `index.html` в браузере.

## Как опубликовать на GitHub Pages

1. Создай новый репозиторий на GitHub.
2. Загрузи в него файлы из этой папки.
3. Открой `Settings` -> `Pages`.
4. Включи `Deploy from a branch`.
5. Выбери ветку `main` и папку `/root`.
6. Сохрани настройки и дождись публикации.

После этого GitHub выдаст публичную ссылку на сайт.
